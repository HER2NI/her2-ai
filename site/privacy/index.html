H.E.R Privacy

What H.E.R Is

H.E.R is a local-first interaction observability lens. Core rendering and telemetry operate in-browser.

What We Do Not Collect

  • No chat content upload to her2.ai for core operation.
  • No remote storage of conversation text in licensing flows.

Optional Localhost Embeddings

If you enable localhost embeddings, text may be sent to the user-configured localhost endpoint (for example, 127.0.0.1). This is user-controlled and local-network scoped.

Licensing Data Processed

  • Device identifier (or device hash).
  • Stripe checkout session ID.
  • License token material used for activation.

Where Data Goes

  • Stripe for payment processing and checkout records.
  • her2.ai licensing API to redeem paid checkout sessions and issue signed license tokens.

Retention

Stripe retains payment and session records under Stripe policies. her2.ai licensing service keeps data minimal and uses Stripe session metadata for redemption status/audit. Licensing sends only device identifier and Stripe session ID; no chat content is transmitted.

User Controls

Wipe now clears local extension/session state including locally stored tokens and device/license keys.
